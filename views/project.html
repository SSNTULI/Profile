   UNIVERSITY PROJECTS

Project: STOP WATCH

Scope:
Design a stopwatch capable of counting up, counting down, pausing, and resetting, without using programmable chips or microcontrollers.

Role:
Designed the 555 timer-based clock circuit and the logic control using a finite state machine to handle second-to-minute carry operations.

Solution:
Implemented a digital stopwatch using a 555 timer IC for clock generation, binary counter ICs for counting, binary-to-decimal decoder ICs, and 7-segment displays for output visualization. The 555 timer provides the clock pulses to the binary counters, which track seconds and minutes. Binary-coded decimal outputs from the counters are decoded into seven discrete signals to drive the 7-segment displays, with FSM logic handling counting, pause, reset, and carry operations.

Tools
Multisim, Finite State Machine design, Digital Circuit Design

Outcome
A fully functional physical stopwatch demonstrating accurate up-count, down-count, pause, and reset operations.

Project: VENUE CAPACITY AND CONDITION MONITORING SYSTEM

Scope:
Design a system to monitor venue occupancy, control lighting, and maintain room temperature at optimal conditions without using programmable chips The system should trigger an alarm if the maximum occupancy is exceeded, automatically adjust lighting to optimal illumination, and maintain the room temperature at 25 °C.

Role:
Designed the people-counting subsystem and its integration with the overall control logic.

Solution:
Implemented a non-programmable solution using infrared LEDs and sensors to detect people entering and exiting the venue. Occupancy counting and control logic were implemented using a finite state machine, with sensor outputs as inputs to manage counting, alarm triggering, and system state. Additional circuits were designed to control lighting dimness based on real-time occupancy and to maintain temperature using analog control mechanisms.

Tools
Multisim, Finite State Machine design, Digital Electronics Design

Outcome
A fully functional physical system capable of real-time occupancy monitoring, lighting adjustment, and temperature control without microcontrollers.

COIN RECOGNITION AND SUM COUNTING

Scope:
Develop a real-time vision-based system to detect South African coins from a continuous video stream, classify each coin by denomination, and compute the cumulative monetary value.

Role:
Sole developer responsible for system architecture, image-processing pipeline, feature extraction, model training, and UI integration.

Solution:
Implemented a real-time video acquisition pipeline using OpenCV with frame-by-frame preprocessing (grayscale conversion, Gaussian filtering, and background normalization). Applied geometric transformations and circular Hough detection to isolate individual coin candidates.

Extracted texture descriptors using Haralick features (contrast, energy, entropy, homogeneity, and correlation) and combined these with geometric attributes (radius, roundness) to form the feature vector. Trained a feed-forward Artificial Neural Network (ANN) using Scikit-learn to classify coins into their respective denominations (10c–R5).

Developed a live inference loop that performs detection, feature extraction, classification, and incremental sum aggregation in real time. Built a Tkinter interface to display the recognized denominations, running totals, and system diagnostics

Tools
Python2, OpenCV, Scikit-learn, Tkinter

Outcome
A fully operational real-time system capable of detecting and classifying South African coins with high accuracy and computing the total sum directly from a live video feed.
